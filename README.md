# Как редактировать

Для начала войти в аккаунт (справа сверху)
- логин и пароль — как на сервере

## Изменить текст статьи

+ Посмотреть на номер в ссылке страницы, которую хотим изменить
  - Например, [shamir.lv/1003](http://shamir.lv/1003) -> **1003**
  - Или [shamir.lv/en/91](http://shamir.lv/en/81) -> **91**

+ Открыть [src/content/articles](src/content/articles)

+ Найти файл с тем же номером (для латышской или английской версии статьи — lv или en)
  - Например, [shamir.lv/1003](http://shamir.lv/1003) -> [1003.md](/src/content/articles/1003.md)
  - Или [shamir.lv/en/91](http://shamir.lv/en/91) -> [91-en.md](/src/content/articles/91-en.md)

+ Нажать на карандаш справа над статьей

+ Часть в начале, отделенную тремя дефисами, **не трогать**

+ Отредактировать текст в _markdown_
  - Заголовки первого, второго, третьего уровня помечаются решетками (напр. # Заголовок первого уровня)
  - *\*курсив\**, **\*\*жирный\*\***
  - Ссылки: \[текст ссылки\](ссылка)
  - Картинки: \!\[описание картинки\](ссылка-на-картинку)
  - Больше [здесь](https://www.markdownguide.org/basic-syntax/)

+ Нажать на зеленую кнопку Commit changes справа над статьей

+ В появившемся окне нажать Commit changes еще раз

## Добавить картинку

+ Открыть [public/images](public/images)

+ Нажать Add file справа сверху (над желтой плашкой) и Upload files

+ Перетащить картинки или нажать choose your files и выбрать

+ Нажать на зеленую кнопку Commit changes

+ Картинка будет доступна по ссылке /public/images/название-картинки, ее можно вставить в статью (см. выше)

## Добавить статью

+ Открыть шаблон статьи [templates/article.md](templates/article.md) и нажать на "копировать" (2 квадратика) справа над статьей, немного левее карандаша

+ Вернуться обратно

+ Выбрать уникальный номер для статьи (например, **2001**)

+ Нажать Add file справа сверху (над желтой плашкой) и Create new file

+ В поле name your file ввести _выбранный-номер.md_ (например, **2001.md**)
  - Для английской или латышской версии — соответственно, **2001-en.md** или **2001-lv.md**)

+ В поле Enter file contents here вставить статью, которую скопировали ранее

+ Отредактировать часть между тремя дефисами
  - id — выбранный нами номер
  - title — заголовок
  - image — ссылка на картинку (см. выше, как загрузить)
  - tags — тэги (см. [shamir.lv/archive](https://shamir.lv/archive))
  - date — дата (до буквы T)
  - locale — язык (ru, en, lv)
  - oldUrl **не трогать**

+ Вставить текст статьи

+ Нажать на зеленую кнопку Commit changes справа над статьей

+ В появившемся окне нажать Commit changes еще раз

+ Сделать то же самое для двух других языков

## Отредактировать текст не-статьи

+ Открыть [src/i18n/translations.ts](src/i18n/translations.ts)

+ Нажать на карандаш справа над статьей

+ Поиском найти текст, который надо отредактировать

+ Отредактировать

+ Нажать на зеленую кнопку Commit changes справа над статьей

+ В появившемся окне нажать Commit changes еще раз

# Общее

- Изменения отображаются на сайте в течение пяти минут

- По умолчанию в редакторе выключен перенос строки. Справа над редактором можно нажать No wrap и выбрать Soft wrap
